# Search engine of ski accomodations
A Python Fast API app that integrate with a stream.

## Features
- Search - returns an ID for a specific search (according payload)
- SearchById - returns the results for specfic search key
- Bonus:
    a) rate limiter
    b) Stateless API

## Dependencies
- Node.js
- MongoDB

## Installation

```bash
$ npm install
```

## Running this server 

```bash
$ npm run start
```
